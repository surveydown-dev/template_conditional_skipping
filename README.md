A template of [conditional skipping](https://surveydown.org/docs/conditional-logic#conditional-skipping) (skip to a page if a condition is true).

To create this template, run this command in your R console:

```r
surveydown::sd_create_survey(
  #path = "path/to/survey",
  template = "conditional_skipping"
)
```

Refer to the [Start with a template](https://surveydown.org/docs/getting-started#start-with-a-template) section for more details.
